# 🏫 FindVoid 빈 강의실 대여 시스템

## 🌟 주요 기능
1. **강의실 시간표 보기 📅**
   - 특정 건물과 강의실을 선택하여 요일별/시간대별 예약 상태를 확인합니다.
   
2. **비어있는 강의실 검색 🔍**
   - 선택한 요일과 시간대에 비어있는 강의실을 순차 탐색 또는 이진 탐색을 통해 찾을 수 있습니다.
   
3. **강의실 예약 신청 📝**
   - 원하는 요일과 시간대에 강의실을 예약할 수 있습니다.
